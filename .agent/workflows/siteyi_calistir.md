# Siteyi Çalıştırma

Siteyi geliştirmek ve önizlemek için geliştirici sunucusunu başlatın.

1. Proje klasörüne gidin (Eğer orada değilseniz):
   ```bash
   cd "/Users/serkann/Desktop/Site Projesi/ataturk-memorial"
   ```

2. Gerekli paketleri yükleyin (İlk kez çalıştırıyorsanız):
   ```bash
   npm install
   ```

3. Geliştirici sunucusunu başlatın:
   ```bash
   npm run dev
   ```

4. Tarayıcınızda şu adresi açın:
   http://localhost:3000
